## 说明

### 1、项目简介

​		静态 & 动态

​		路径规划 & 轨迹规划

### 2. 环境配置

1. 运行环境： `python3`
2. 使用的库： `numpy, matplotlib, pandas`
3. 库的安装： `pip install -i`

### 3. 运行命令

​	运行：

* `python3 static_PID.py`  静态避障 `RRT_smart + PID`
* `python3 static_SC.py`  静态避障 `RRT* + Smooth control`

* `python3 dynamic_PID.py`  动态避障 `RRT+PID`
